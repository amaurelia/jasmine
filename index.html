¿Qué es Jasmine?

Jasmine es un framework de pruebas unitarias para JavaScript que nos permite:

  • Escribir pruebas legibles
  • Verificar el comportamiento del código
  • Detectar errores tempranamente
  • Mejorar la calidad del software
Funciones de Ejemplo

Calculadora:

  • sumar(a, b)
  • restar(a, b)
  • multiplicar(a, b)
  • dividir(a, b)

Utilidades:

  • esPar(numero)
  • formatearNombre(nombre)
  • validarEmail(email)
Cobertura de Código

La cobertura de código mide qué porcentaje de nuestro código está siendo probado por las pruebas unitarias.

Tip: Una buena cobertura está entre 80-90%
